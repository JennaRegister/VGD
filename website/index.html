Welcome!

This site contains details about the course, daily lesson plans, and code for you to check out! The goal of this course is to get an understanding and appreciation for what goes in to making a video game. We will make multiple small arcade-style games (sorry - no big 3D blockbusters!) We will be using Python 3 and PyGame.
